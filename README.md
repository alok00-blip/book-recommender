# 📚 Book Recommender AI

A personalized book recommendation chatbot powered by Agentic AI.

## What it Does
Tell the chatbot your mood, interests, or goals and it will recommend 
the perfect book for you — with Amazon India buy links!

## Examples
- "I am feeling stressed after work, suggest a book"
- "I want to become a better leader"
- "I love mystery novels, what should I read next?"
- "Suggest a book for my 10 year old who loves adventure"

## Built With
- Google Agent Development Kit (ADK)
- Gemini 2.5 Flash
- Google Cloud Vertex AI
- Streamlit
- Python

## How it Works
The app uses a multi-agent architecture:
- **Root Agent** — understands your mood and coordinates recommendations
- **Google Search Agent** — searches the web for current book information
- **URL Context Agent** — reads book details from websites
